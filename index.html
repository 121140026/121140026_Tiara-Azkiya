Tentang Website

Assalamualaikum, Perkenalkan nama saya Tiara Azkiya dengan nim 121140026 dari kelas Pemrograman Web RA. Ini adalah website yang berisi : Halaman Utama, Halaman Formulir, Halaman Tabel.

Halaman Utama berisi : Judul halaman yang sesuai, Menu navigasi untuk berpindah ke halaman lain (Form dan Tabel), Deskripsi singkat tentang website

Halaman Formulir berisi :

  1. Buatlah form pendaftaran dengan field: Nama, NIM, Program Studi, Email, Password, Jenis Kelamin, Tanggal Lahir, dan Alamat.
  2. Setiap field harus memiliki label yang jelas.
  3. Gunakan atribut required untuk field yang wajib diisi.
  4. Tambahkan tombol submit di bagian bawah form.
  5. Menggunakan method get dan mendesain query string yang akan dikirim

Halaman Tabel berisi :

  1. Buatlah tabel yang menampilkan daftar mahasiswa dengan kolom: Nama, NIM, Program Studi, Email, Password, Jenis Kelamin, Tanggal Lahir, dan Alamat.
  2. Tabel harus memiliki header dan setidaknya 5 baris data (menggunakan data dummy).
  3. Gunakan styling CSS untuk memberikan warna latar belakang yang berbeda pada baris genap (even) dan ganjil (odd)
